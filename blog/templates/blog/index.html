    ezPy



    A site for users to create and save their ideal Python development environment. Once created, users can share or download their customized environment as a Python script that will install and setup each customization on their local machine.
    Built by a team of five developers over four days using Python and the Django framework.

    Dine-o-Truck



    A site for Seattle to find food trucks in their neighborhoods, complete with details such as hours, payment, addresses, and social network links to their favorite trucks.
    Built by a team of three developers over four days using Python, Pyramid and SQLAlchemy.
